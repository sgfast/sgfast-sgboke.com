--------------------------------------------------------------------------------
// 导航条
--------------------------------------------------------------------------------
.navbar				: 标识一个导航条
.navbar-default		: 默认的导航条颜色
.navbar-inverse		: 反色导航条
.navbar-brand		: 标识logo
.navbar-header		: 标识标题
.navbar-form		: 标识一个导航条内的表单
.navbar-left		: 左浮动
.navbar-right		: 右浮动
.navbar-nav			: 导航条中的ul
.navbar-btn			: 导航条中的按钮
.navbar-text		: 导航条中的文字
.navbar-link		: 导航条中的链接
.navbar-fixed-top	: 固定在顶部
.navbar-fixed-bottom: 固定在底部


--------------------------------------------------------------------------------
// 导航
--------------------------------------------------------------------------------
.nav				: 导航的基本样式
.nav-tabs			: 标签型导航样式
.nav-pills			: 胶囊型导航样式
.nav-stacked		: 垂直导航
.nav-divider		: 垂直导航分隔线
.nav-justified		: 自适应导航，需要与nav-tabs或nav-pills一起使用(屏幕需大于768)
.breadcrumb			: 面包屑导航包裹类，在其下的active类可以标识最后一个元素


--------------------------------------------------------------------------------
// 列表
--------------------------------------------------------------------------------
.list-styled		: 列表无样式
.list-inline		: 列表一行内显示
.dl-horizontal		: 水平定义列表（只有大于768时才有用）


--------------------------------------------------------------------------------
// 按钮
--------------------------------------------------------------------------------
.btn				: 基本按钮样式
.btn-default		: 按钮基本风格
.btn-primary		: 主要蓝色风格
.btn-success		: 绿色成功风格
.btn-warning		: 桔黄警告风格
.btn-danger			: 红色错误风格
.btn-lg				: 大号风格
.btn-sm				: 小号风格
.btn-xs				: 超小号风格
.btn-block			: 块状按钮风格（全屏显示）


--------------------------------------------------------------------------------
// 按钮组
--------------------------------------------------------------------------------
.btn-group			: 标识div用于包裹多个btn元素，形成一个组(btn-group可以和dropdown通用)
.btn-group-lg		: 大按钮组
.btn-group-sm		: 小按钮组
.btn-group-xs		: 超小按钮组
.btn-group-vertical	: 纵向btn组
.btn-group-justified: 等分按钮组
.dropup				: 向上弹起下拉菜单
.btn-toobar			: 将此类放置于btn-group之外，就可以让多个group形成一个toolbar


--------------------------------------------------------------------------------
// 图片
--------------------------------------------------------------------------------
.img-responsive		：响应式图片，主要针对于响应式设计
.img-rounded		：圆角图片
.img-circle			：圆形图片
.img-thumbnail		：缩略图片


--------------------------------------------------------------------------------
// 图标
--------------------------------------------------------------------------------
.glyphicon			: 将i和span标签添加glyphicon即可将此标签设置成iconfont，其它类可查看图集


--------------------------------------------------------------------------------
// 表单
--------------------------------------------------------------------------------
.form-horizontal	: 水平表单，但实际上只有大屏才能水平显示
.form-inline		: 内联静音，也是在大屏下才能内联显示
.form-group			: 放在外圈div，使其包含label和input，成为一个input容器
.has-success		: 放在外圈div，成功状态
.has-error			: 放在外圈div，错误状态
.has-warning		: 放在外圈div，警告状态
.has-feedback		: 放在外圈div，使input带有一个警示图标（这个图标必须手工用span或i实现出来）
.form-control		: 标示此控件是一个bootstrap的form控件，会得到基础样式
.checkbox			: 将一个div标识为checkbox，注意是用在包圈div之上
.radio				: 将一个div标识为radio，注意是用在包圈div之上
.input-sm			: 比普通控件大小更小一些
.input-lg			: 比普通控件大小更大一些
.help-block			: 提示信息以块状显示，显示在控件底部
.help-inline		: 提示信息以内联显示，显示在控件右侧


--------------------------------------------------------------------------------
// 表格
--------------------------------------------------------------------------------
.table				: 表格基础
.table-striped		: 斑马线表格
.table-bordered		: 带边框的表格 
.table-hover		: 鼠标悬停高亮的表格
.table-condensed	: 紧凑型表格
.table-responsive	: 响应式表格

.active				: tr : 表示当前活动的信息
.success			: tr : 表示成功或者正确的行为
.info				: tr : 表示中立的信息或行为
.warning			: tr : 表示警告，需要特别注意
.danger				: tr : 表示危险或才可能是错误的行为


--------------------------------------------------------------------------------
// 标签,徽章
--------------------------------------------------------------------------------
.label				: 标识一个标签
.label-default		: 默认样式
.label-primary		: 主要样式
.label-success		: 成功标签
.label-info			: 信息标签
.label-warning		: 警告标签
.label-danger		: 错误标签

.badge				: 标识一个徽章


--------------------------------------------------------------------------------
// 分页
--------------------------------------------------------------------------------
.pagination			: 用于标识分页条
.pagination-lg		: 大分页条
.pagination-sm		: 小分页条
.pager				: 标识小导航条（即上页下页）
.previous			: 标识于li，表明上一页，左对齐
.next				: 标识于li，表明下一页，右对齐


--------------------------------------------------------------------------------
// 代码
--------------------------------------------------------------------------------
<code>				: 主要用于显示单行内联代码
<pre>				: 主要用于显示多行块级代码
<kbd>				: 主要用于展示需要用户输入的代码
.pre-scrollable		: 用于pre，当写入该类时，pre框的最大高度为340

--------------------------------------------------------------------------------
// 下拉菜单(需js)
--------------------------------------------------------------------------------
.dropdown			: 此div容器包裹整个下拉菜单的元素(注意，必须使用它包裹)
.dropdown-toggle	: 用于标识弹出菜单的按钮，此按钮必须带有data-toggle属性，并且必须命名为：dropdown
.dropdown-menu		: 用于标识ul下拉菜单的class
.dropdown-menu-right: 标识ul下拉菜单与button为右对齐
.pull-right			: 同上作用
.divider			: 用于标识menu内部的li为一条分隔线
.dropdown-header	: 用于标识menu内部的li为下拉菜单的标题（是无法选择的）
